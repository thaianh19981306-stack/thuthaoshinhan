# Website Thu Thảo Shinhan

Website tĩnh dùng cho GitHub Pages.

## Cách đăng
1. Upload toàn bộ file trong thư mục này lên repository GitHub.
2. Vào Settings → Pages.
3. Chọn Deploy from a branch.
4. Chọn branch `main`, thư mục `/ (root)`.
5. Bấm Save.
